Why Choose Flutter?

Flutter is Google's UI toolkit for building beautiful, natively compiled applications for mobile, web, and desktop from a single codebase.

Fast Development

Hot reload allows you to quickly build UIs, add features, and fix bugs faster with instant preview.

Expressive UI

Create beautiful, customized user interfaces with Flutter's rich set of widgets and Material Design.

Single Codebase

Write once, deploy everywhere - to mobile, web, desktop, and embedded devices with the same code.

Installation Guide

Follow these steps to install Flutter on your system and set up your development environment.

System Requirements

Before installing Flutter, make sure your system meets the following requirements:

  • Operating System: Windows 7 or later (64-bit), macOS (64-bit), or Linux (64-bit)
  • Disk Space: 1.32 GB (does not include disk space for IDE/tools)
  • Tools: Git for Windows, if you're installing on Windows

Installation Steps

  1. Download the Flutter SDK from the official website
  2. Extract the file to your desired location
  3. Add the Flutter tool to your path
  4. Run flutter doctor to check for any dependencies you need to install
  5. Install any missing dependencies as prompted

Essential Flutter Commands

Master these essential Flutter CLI commands to streamline your development workflow.

Command Description Usage Example
flutter create Create a new Flutter project flutter create my_app
flutter run Run your Flutter app on an attached device or emulator flutter run
flutter build Build a release version of your app flutter build apk
flutter pub get Get packages in your project flutter pub get
flutter doctor Show information about the installed tooling flutter doctor
flutter clean Delete the build/ and .dart_tool/ directories flutter clean
flutter test Run Flutter unit tests flutter test
flutter devices List all connected devices flutter devices
flutter analyze Analyze the project's Dart code flutter analyze
flutter attach Attach to a running Flutter app flutter attach
flutter channel List or switch Flutter channels flutter channel stable
flutter upgrade Upgrade Flutter to the latest version flutter upgrade
flutter downgrade Downgrade Flutter to the previous version flutter downgrade
flutter emulators List and launch emulators flutter emulators --launch Pixel_3a_API_30
flutter install Install app on a connected device flutter install
flutter logs Show log output for running Flutter apps flutter logs
flutter format Format Dart code according to style guidelines flutter format .
flutter drive Run Flutter driver tests flutter drive --target=test_driver/app.dart
flutter gen-l10n Generate localization files flutter gen-l10n
flutter assemble Assemble and build Flutter resources flutter assemble
# Create a new Flutter project
flutter create my_app

# Navigate to the project directory
cd my_app

# Run the app on an available device
flutter run
